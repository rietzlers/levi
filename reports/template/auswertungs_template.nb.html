Numerische Analyse

Wie im Plot angedeutet wird wird ein linearer und ein quadratischer Fit durchgeführt um zu überprüfen ob die numerische Analyse die Vermutung eines nichtlinearen Zusammenhanges bestätigt.

Die Daten werden so umskaliert, das der Temperatur-Nullpunkt der Schmelztemperatur von CMSX10 (?°C) entspricht. Die bei ‘Intercept’ angegebene Größe entspricht damit der Viskosität am Schmelzpunkt.

T_melt = 1388

  select(-(X1))

st_raw <- read_csv("/home/stephan/workspace/Projekt_Metalllegierungsschmelzen/Daten/ISS/MC2/MC2_eval_cycle_1_bis_6/MC2_cycle_1_bis_6_surface_tension_results.csv") %>%  select(-(X1))

visc %>%
